## Movies App

The **Movies App** helps users find and explore movies. It allows searching by title, viewing details like ratings, cast, trailers, and more. You can also add movies to a watchlist for later viewing.

### Features:
- **Search for Movies**: Type a movie title and view suggestions.
- **Movies Grid Display**: View multiple movies with posters, titles, and release dates.
- **Movie Details**: View synopsis, ratings, cast, trailers, and more.
- **Watchlist**: Save movies to a watchlist, stored in local storage.
